JSHunter Secrets Test Page

Esta página inclui arquivos JavaScript que contêm secrets falsos para testar a extensão JSHunter Burp.

Quando você navegar por esta página com o Burp Suite interceptando o tráfego, a extensão JSHunter deve:

  1. Detectar os arquivos app.js e vendor.js.
  2. Escaneá-los com o TruffleHog.
  3. Encontrar os secrets falsos.
  4. Exibir os resultados na aba JSHunter no Burp Suite.
  5. Enviar uma notificação para seu Discord (se configurado).

Scripts Carregados

  • scripts/app.js: Contém uma chave de acesso AWS falsa.
  • scripts/vendor.js?version=1.2.3: Contém um token do GitHub falso.
  • Script inline: Contém uma chave de API do Google falsa.

Como Usar

Siga os passos no arquivo COMO_USAR_TESTE.md que criei para você.
